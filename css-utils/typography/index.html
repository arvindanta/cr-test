# Typography

We can easily add font related styles through SASS mixin and CSS Utils.

# Heading

For heading text, we can use 'fw-type-#{size}'.

#{size} can be 'h1', 'h2' .. 'h6', 'h7'

# Heading size chart:

rem 2.75rem 2rem 1.5rem 1.125rem 1rem 0.875rem 0.75rem
